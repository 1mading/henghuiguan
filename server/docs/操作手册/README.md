# 恒慧管 · 操作手册（按角色）

> 部门内部项目管控系统 · 正式环境从钉钉工作台打开  
> 访问地址示例：`https://henghuiguan.handagroup.com/app`

---

## 角色说明

| 角色 | 系统标识 | 定位 | 数据范围 |
|------|----------|------|----------|
| **总经理** | `gm` | 全局统筹、审核决策 | 全公司所有项目与任务 |
| **管理员** | `admin` | 系统运维、与总经理同等权限 | 全公司所有项目与任务 |
| **部门经理** | `manager` | 本部门项目落地、任务分配与监督 | 仅本部门项目及下属任务 |
| **执行人员** | `staff` | 任务执行、进度反馈 | 仅本人负责/协办的任务与相关项目 |

> **说明**：总经理与管理员在系统中权限相同（`isFullAccess`），菜单与操作一致，区别仅在岗位职责。

**最近更新（2026-06）**：任务留言讨论（@ 提及 + 钉钉提醒）、附件/留言 Ctrl+V 粘贴图片、图片点击放大预览；部门经理人员档案（本部门有限编辑）；团队饱和度按当周统计。
