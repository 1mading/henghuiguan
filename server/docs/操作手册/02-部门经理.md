# 部门经理 操作手册

> 适用角色：`manager`（部门经理）  
> 权限范围：**全公司项目**只读浏览；**管理权**取决于是否为项目负责人/创建人，而非部门归属

---

## 1. 您能看到什么

- **全公司** 所有项目（列表、详情、全部任务），与总经理列表范围一致
- **管理权**：仅本人为 **项目负责人/创建人** 的项目可编辑、归档、添加任务、上传文档；其他项目 **只读**
- 本部门成员的团队饱和度（团队管理页仅本部门数据）
- 本部门相关操作日志
- 工作台、任务中心仍只展示 **您本人** 负责/协办的任务

> **重要**：部门经理身份 **不等于** 可管理本部门所有项目。若需管理某项目，请确保项目详情中 **项目负责人** 为您，或由总经理/管理员调整。

---

## 2. 菜单说明

| 菜单 | 说明 |
|------|------|
| **团队管理** | 本部门成员饱和度、成员任务看板 |
| **工作台** | 本人任务概览 + 协办专区 |
| **项目管理** | 全公司项目浏览；新建项目；本人负责项目的编辑/归档 |
| **任务中心** | **我的**：本人负责/协办末级任务；**部门**：本部门成员负责的末级任务（监督用） |
| **人员档案** | 查看 **本部门** 人员，有限编辑执行人员信息 |
| **操作日志** | 本部门相关变更记录 |
| **归档管理** | 查询已归档项目（本人负责项目可在项目详情归档） |
| **数据导入** | 批量导入（补全种子数据仅总经理/管理员） |
| **快速建任务** | 临时任务 |

> **人员档案**：可查看本部门人员，编辑执行人员的 **职位、直属上级、标准周工时**。**不可**同步钉钉通讯录、添加人员或修改角色/部门/姓名；人员绑定请联系总经理/管理员。

---

## 3. 典型操作流程

### 3.1 维护本部门人员档案

1. **人员档案** → 查看本部门成员列表
2. 对 **执行人员** 可编辑：职位、直属上级、标准周工时
3. 姓名、部门、角色为只读；同步钉钉、添加人员需联系总经理/管理员

### 3.2 新建项目

1. **项目管理** → **新建项目**
2. 填写项目名称、描述、所属部门、**项目负责人**（建议设为您本人以便后续管理）
3. 可选指定 **团队成员**（协助推进任务；成员可编辑/留言/上传附件，但不可拆分任务或归档项目）
4. 指定计划起止日期 → 保存后进入详情，上传 **项目文档**（可选）

> 任何部门经理均可新建项目，不限于本部门；但仅能 **管理** 本人为负责人/创建人的项目。

### 3.3 拆分与分配任务

1. 进入 **本人负责** 的项目详情 → **添加任务**
2. 填写任务标题、负责人（可选本部门及其他部门执行人）、计划开始、预计工时、**优先级**；带教类可填 **每日投入工时**
3. 可选填 **告知性/辅助性协办人**（辅助性须协办人填时段并经您或负责人审批）
4. 需要进一步分解时 → 任务详情 → **添加子任务**
5. 保存后负责人收到钉钉 **新任务** 通知

**工时提示**：子任务预计工时总和不能超过父任务；分配外部部门人员时会标注外部工时。

### 3.4 跨项目依赖

当任务须等待 **其他项目** 任务完成后才能推进：

1. 打开后继任务详情 → **依赖关系（跨项目）** → **添加前置依赖**
2. 选择其他项目的前置任务，填写说明
3. 项目详情可查看 **跨项目 / 任务依赖** 汇总表
4. 阻塞期间任务不占团队成员当周饱和度

> 须为任务可编辑人（负责人/协办人/创建人/项目负责人）方可添加/解除依赖。

### 3.5 过程监督

#### 团队管理

1. 查看本部门每人：进行中数、预计/实际工时、**当周**饱和度（含已批准辅助性协办工时）
2. 按饱和度排序，识别 overload 成员
3. 点击成员 → 查看其任务看板
4. 顶部 **导出** 可下载本部门团队 Excel

> 监督他人任务进度：进入 **项目管理** 找到对应项目查看；**任务中心** 仅显示您自己的任务。

#### 任务详情干预（须为项目负责人）

| 操作 | 场景 | 要求 |
|------|------|------|
| **驳回** | 交付物不合格、需返工 | 必填驳回原因，推送负责人 |
| **转办** | 调整执行人 | 必填转办原因，推送新负责人 |
| **快捷更新进度** | 本人可编辑的任务 | 滑块调整 0–100% |
| **编辑** | 修改任务字段 | 须为项目负责人/创建人或任务相关人 |
| **作废** | 任务不再执行 | 须为项目负责人或任务创建人 |
| **审批辅助性协办** | 协办人提交协助时段 | 任务详情中同意/拒绝 |

> 对 **非本人负责项目** 下的任务，您只能 **只读查看**，不能驳回/转办/编辑。可联系该项目负责人或总经理/管理员处理。

#### 任务留言

- 任务详情 **留言讨论**：须为任务相关人（负责人/协办人/项目负责人等）方可留言
- 输入 **@** 提及相关人员，对方会收到钉钉 **留言提及** 通知（需已绑定 userid）
- 留言区 **Ctrl+V 粘贴图片**；任务附件区点击后也可 **Ctrl+V** 粘贴截图
- 可 **添加钉钉文档链接** 作为项目/任务附件

### 3.6 项目收尾

1. 确认下属任务均已 **已完成**（在本人负责的项目下操作）
2. 项目详情 → 推进项目状态为 **已完成**
3. 点击 **归档** → 项目移入归档管理，变为只读

### 3.7 临时任务

侧边栏 **快速建任务**：

- 不关联项目，适合部门内零星事项
- 指定负责人（默认可选本部门人员）
- 同样计入任务中心与工时统计

### 3.8 导出数据

- **项目管理** 页 → 导出当前筛选条件下的项目 Excel（默认含全公司，可按部门筛选）
- **任务中心** 页 → 导出 **本人** 任务 Excel
- **团队管理** 页 → 导出本部门团队饱和度 Excel

## 4. 权限边界

| 可以做 | 不可以做 |
|--------|----------|
| **浏览** 全公司所有项目与任务 | 编辑/归档 **非本人负责** 的项目 |
| 新建项目（任意部门） | 删除项目（仅总经理/管理员） |
| 在 **本人负责项目** 下添加主/子任务、指定协办人/团队成员 | 替他人项目管理任务（非项目负责人时） |
| **驳回/转办/作废** 本人负责项目下的任务 | 对非本人负责项目任务进行干预 |
| 配置/解除本人可编辑任务的前置依赖 | 同步钉钉通讯录、使用滚动大屏 |
| 查看/有限编辑本部门 **人员档案** | 添加人员、修改角色/部门/姓名 |
| 团队管理（本部门饱和度） | 查看其他部门操作日志（服务端按范围过滤） |
| 任务留言、@ 提及、粘贴图片、钉钉文档链接 | — |

---

## 5. 推送说明

| 场景 | 您是否收到 |
|------|------------|
| 下属被分配新任务 | 否（通知负责人本人） |
| 下属任务被驳回/转办 | 否（通知当事人） |
| 下属 **完成任务** | **是**（若您是执行人直属上级 `leaderId`） |
| 您负责的项目有任务完成 | **是**（若您是项目负责人） |
| 辅助性协办待审批 | **是**（若您是任务负责人或项目负责人） |
| 任务临期/逾期 | 否（通知任务负责人；您是负责人时才会收到） |
| 留言中被 @ 提及 | **是**（若您为负责人/协办人/项目负责人且已绑定 userid） |

## 6. 常见问题

**Q：我是部门经理，为什么不能编辑某个本部门项目？**  
A：项目管理权看 **项目负责人/创建人**，不看部门。请确认项目详情中负责人是否为您，或联系总经理调整。

**Q：为什么我不能编辑某个任务？**  
A：须为任务负责人、协办人、创建人或 **该项目负责人**。其他情况只能只读查看，或请项目负责人/管理员协助。

**Q：任务中心为什么看不到下属的任务？**  
A：请切换到任务中心顶部的 **「部门」** 页签，可查看本部门成员负责的末级任务。也可使用 **团队管理** → 点击成员查看其任务看板。

**Q：执行人说完成了但没通知我？**  
A：完成通知发给 **项目负责人** + **执行人直属上级**。请确认项目详情中负责人是否为您，以及人员档案中上下级关系是否正确。

**Q：子任务工时超了怎么办？**  
A：调整子任务预计工时，或增大父任务工时后再分配。系统保存时会校验并阻止超额。

**Q：如何查看已归档项目？**  
A：**归档管理** 菜单查询；新归档请在 **本人负责** 的项目详情操作。
