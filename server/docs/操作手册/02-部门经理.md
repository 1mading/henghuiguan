# 部门经理 操作手册

> 适用角色：`manager`（部门经理）  
> 权限范围：**全公司项目**只读浏览；本部门人员与任务 **管理权** 按项目负责人/创建人规则

---

## 1. 您能看到什么

- **全公司** 所有项目（列表、详情、全部任务），与总经理列表范围一致
- **管理权**：仅本人为 **项目负责人/创建人** 的项目可编辑、归档、添加任务；其他项目只读
- 本部门成员的操作日志与团队饱和度
- 工作台仍只展示 **您本人** 负责/协办的任务

---

## 2. 菜单说明

| 菜单 | 说明 |
|------|------|
| **团队管理** | 本部门成员饱和度、成员任务看板 |
| **工作台** | 本人任务概览 |
| **项目管理** | 本部门项目的新建/编辑/归档 |
| **任务中心** | 本部门全部任务 |
| **人员档案** | 查看 **本部门** 人员，有限编辑执行人员信息 |
| **操作日志** | 本部门相关变更记录 |
| **归档管理** | 本部门已归档项目 |
| **数据导入** | 本部门批量导入 |
| **快速建任务** | 临时任务 |

> **人员档案**：可查看本部门人员，编辑执行人员的 **职位、直属上级、标准周工时**。**不可**同步钉钉通讯录、添加人员或修改角色/部门/姓名；人员绑定请联系总经理/管理员。

---

## 3. 典型操作流程

### 3.1 维护本部门人员档案

1. **人员档案** → 查看本部门成员列表
2. 对 **执行人员** 可编辑：职位、直属上级、标准周工时
3. 姓名、部门、角色为只读；同步钉钉、添加人员需联系总经理/管理员

### 3.2 新建本部门项目

1. **项目管理** → **新建项目**
2. 所属部门默认为本部门（或选本部门）
3. 指定项目负责人（通常为您或部门骨干）
4. 保存后进入详情，上传 **项目文档**（可选）

### 3.3 拆分与分配任务

1. 项目详情 → **添加任务**
2. 填写任务标题、负责人（可选本部门及其他部门执行人）、计划开始、预计工时
3. 需要进一步分解时 → 任务详情 → **添加子任务**
4. 保存后负责人收到钉钉 **新任务** 通知

**工时提示**：子任务预计工时总和不能超过父任务；分配外部部门人员时会标注外部工时。

### 3.4 过程监督

#### 团队管理

1. 查看本部门每人：进行中数、预计/实际工时、**当周**饱和度
2. 按饱和度排序，识别 overload 成员
3. 点击成员 → 查看其任务看板，必要时调整分配

#### 任务详情干预

| 操作 | 场景 | 要求 |
|------|------|------|
| **驳回** | 交付物不合格、需返工 | 必填驳回原因，推送负责人 |
| **转办** | 调整执行人 | 必填转办原因，推送新负责人 |
| **快捷更新进度** | 本部门任务（含非本人负责） | 滑块调整 0–100% |
| **编辑** | 修改任务字段 | 若您非负责人，部分字段可能受限；负责人编辑需填修改原因 |

> 部门经理对 **本部门项目下任务** 可驳回、转办、更新进度；**标记完成/开始执行** 通常需为任务负责人，或由总经理/管理员操作。

#### 任务留言

- 任务详情 **留言讨论**：本部门经理、负责人、协办人、项目负责人均可留言
- 输入 **@** 提及相关人员，对方会收到钉钉 **留言提及** 通知（需已绑定 userid）
- 留言区 **Ctrl+V 粘贴图片**；任务附件区点击后也可 **Ctrl+V** 粘贴截图
- 点击留言或附件中的图片可 **放大预览**

### 3.5 项目收尾

1. 确认下属任务均已 **已完成**
2. 项目详情 → 推进项目状态为 **已完成**
3. 点击 **归档** → 项目移入归档管理，变为只读

### 3.6 临时任务

侧边栏 **快速建任务**：

- 不关联项目，适合部门内零星事项
- 指定负责人（默认可选本部门人员）
- 同样计入任务中心与工时统计

### 3.7 导出数据

- **项目管理** 页 → 导出本部门项目 Excel
- **任务中心** 页 → 导出本部门任务 Excel
- **团队管理** 页 → 导出本部门团队饱和度 Excel

## 4. 权限边界

| 可以做 | 不可以做 |
|--------|----------|
| 新建/编辑/归档 **本部门** 项目 | 查看或编辑其他部门项目 |
| 在本部门项目下 **添加主/子任务** | 在执行人员无项目权限时替其拆任务 |
| **驳回/转办** 本部门相关任务 | 同步钉钉通讯录 |
| **作废** 本部门人员 **创建** 的任务 | 添加人员、修改角色/部门/姓名 |
| 更新本部门任务进度 | 查看其他部门操作日志 |
| 查看/有限编辑本部门 **人员档案** | 编辑其他部门人员 |
| 任务留言、@ 提及、粘贴图片 | — |
| 上传本部门项目文档、督促上传任务附件 | — |

---

## 5. 推送说明

| 场景 | 您是否收到 |
|------|------------|
| 下属被分配新任务 | 否（通知负责人本人） |
| 下属任务被驳回/转办 | 否（通知当事人） |
| 下属 **完成任务** | **是**（若您是执行人直属上级 `leaderId`） |
| 您负责的项目有任务完成 | **是**（若您是项目负责人） |
| 任务临期/逾期 | 否（通知任务负责人；您是负责人时才会收到） |
| 留言中被 @ 提及 | **是**（若您为负责人/协办人/项目负责人且已绑定 userid） |

## 6. 常见问题

**Q：为什么我不能编辑某个任务？**  
A：「编辑」按钮对负责人和总经理/管理员开放。您可用 **驳回/转办/快捷进度** 干预；或联系管理员协助编辑。

**Q：执行人说完成了但没通知我？**  
A：完成通知发给 **项目负责人** + **执行人直属上级**。请确认项目详情中负责人是否为您，以及人员档案中上下级关系是否正确。

**Q：子任务工时超了怎么办？**  
A：调整子任务预计工时，或增大父任务工时后再分配。系统保存时会校验并阻止超额。

**Q：如何查看已归档项目？**  
A：**归档管理** 菜单，仅查询；新归档请在项目详情操作。
