# 执行人员 操作手册

> 适用角色：`staff`（执行人员）  
> 权限范围：本人负责/协办的任务可编辑；**全公司项目**在项目管理中只读浏览

---

## 1. 您能看到什么

- **项目管理**：可查看 **全公司** 所有未归档项目（列表、详情、项目下 **全部** 任务树）
- **任务详情**：可打开任意项目下的任务详情，**只读** — 无编辑/转办/驳回等（除非您是负责人/协办人/项目负责人）
- **编辑权限**：仅本人为 **主负责人/协办人/项目负责人/创建人** 的任务与项目可编辑
- **工作台**：显示「全公司项目」统计
- **任务中心**：仍只展示本人负责/协办的任务（不变）
- **操作日志**：与本人相关的变更记录

**看不到**：其他同事的临时任务（无项目归属）、团队管理、人员档案、归档管理、数据导入。

---

## 2. 菜单说明

| 菜单 | 用途 |
|------|------|
| **工作台** | 每日入口，查看待办与告警 |
| **任务中心** | 管理本人全部任务 |
| **我的项目** | 查看任务所属项目背景信息 |
| **操作日志** | 查看本人操作历史 |
| **快速建任务** | 创建临时任务（可指派自己或他人） |

## 3. 典型操作流程

### 3.1 每日开始：看工作台

1. 打开恒慧管 → **工作台**
2. 关注顶部红色 **逾期**、黄色 **临期** 提示
3. 点击任务卡片进入详情处理

### 3.2 处理常规任务

1. **任务中心** 或工作台 → 点击任务
2. 查看：所属项目、计划时间、任务描述、上级任务路径
3. 按状态操作：

| 当前状态 | 您可执行 |
|----------|----------|
| 待开始 | **开始执行**（记录实际开始日期） |
| 进行中 | 拖动 **进度滑块**；无子任务时可 **标记完成** |
| 已驳回 | 查看驳回原因 → **编辑** 整改 → 重新 **开始执行** |

4. 上传 **任务附件**（交付截图、文档等）：点击附件区后 **Ctrl+V** 可粘贴图片
5. 在 **留言讨论** 中与项目负责人、协办人沟通：输入 **@** 提及对方，**Ctrl+V** 粘贴过程截图
6. 完成后点击 **标记完成** → 系统通知 **项目负责人** 与 **您的直属上级**

> 若任务有 **未完成的子任务**，您不能直接完成上级任务（需先完成子任务或由经理处理）。

### 3.3 任务留言与 @ 提醒

当您为任务 **负责人** 或 **协办人** 时：

1. 任务详情 → **留言讨论**
2. 输入说明文字；输入 **@** 选择负责人、协办人或项目负责人
3. **Ctrl+V** 粘贴截图作为过程说明（发送前点击缩略图可放大预览）
4. 发送后，被 @ 且已绑定钉钉 userid 的人员会收到 **留言提及** 通知

> 留言图片不会进入任务附件列表；正式交付物请仍上传到 **任务附件**。

### 3.4 编辑自己的任务

1. 任务详情 → **编辑**
2. 可修改：标题、描述、计划开始、预计工时、进度、状态等
3. **必须填写修改原因** → 保存

> 不能修改负责人（需请经理 **转办**）；不能新建项目下主任务/子任务（需请经理拆分）。

### 3.5 任务转办

当无法继续执行时：

1. 任务详情 → **转办**
2. 选择本部门或其他部门同事（按列表可选范围）
3. **必填转办原因** → 确认
4. 新负责人收到钉钉通知

> 仅 **本人负责** 的任务可发起转办（经理/管理员也可代转）。

### 3.6 快速建临时任务

适用于不在正式项目内的零星工作：

1. 侧边栏 **快速建任务**（或手机底栏「新建」）
2. 填写标题、描述、负责人（默认自己）
3. 可选填计划时间与优先级
4. 保存 → 出现在任务中心「临时任务」标签

### 3.7 查看项目背景

1. **我的项目** → 点击项目
2. 可查看项目描述、进度、任务树
3. **不能** 添加项目任务、归档项目、上传项目文档

### 3.8 导出本人任务

**任务中心** 页顶部 → **导出** → 下载 Excel 任务清单。

## 4. 权限边界

| 可以做 | 不可以做 |
|--------|----------|
| 更新 **本人负责** 的任务进度与状态 | 新建/编辑 **项目** |
| 编辑本人任务（填修改原因） | 添加项目 **主任务/子任务** |
| 转办本人任务 | **驳回** 他人任务 |
| 上传本人任务附件（含 Ctrl+V 粘贴图片） | **作废** 任务（需经理/管理员） |
| 任务留言、@ 提及、粘贴图片（负责人/协办人） | 查看团队管理、他人任务 |
| 创建 **临时任务** | 数据导入、人员档案、钉钉同步 |
| 导出本人任务列表 | 修改项目文档 |
| 查看参与项目的只读信息 | — |

---

## 5. 推送说明

| 事件 | 您是否收到 |
|------|------------|
| 被分配新任务 | ✓ |
| 任务被转办给您 | ✓ |
| 任务被驳回 | ✓ |
| 任务即将到期 / 已逾期 | ✓（您是负责人时） |
| 您自己完成任务 | ✗（通知项目负责人和您的上级） |
| 留言中被 @ 提及 | ✓（您是负责人/协办人/项目负责人时） |
| 同事完成任务 | ✗ |

未收到推送：联系管理员确认人员档案已绑定钉钉 userid。

---

## 6. 工时与截止日期

- 填写 **计划开始** + **预计工时** 后，系统自动计算 **计划截止日期**（按公司工作日历）
- **开始执行** 后记录实际开始日；**标记完成** 后自动计算 **实际工时**
- 子任务若由经理分配，工时不能超过上级任务剩余额度

## 7. 常见问题

**Q：为什么我没有「添加任务」按钮？**  
A：执行人员不能拆分项目任务。请联系部门经理在项目下创建并分配给您。

**Q：任务被驳回了怎么办？**  
A：查看详情顶部驳回原因 → 编辑修改 → 保存（填原因）→ 重新开始执行。

**Q：完成了为什么我没收到钉钉？**  
A：完成通知发给 **项目负责人** 和 **您的直属上级**，不发给执行人本人。若您兼项目负责人则会收到。

**Q：进度滑块拖不动？**  
A：可能该任务有未完成的子任务，或任务已结束/归档。请完成子任务或联系经理。

**Q：临时任务和常规任务有什么区别？**  
A：临时任务不挂项目，您可自行创建；常规任务由经理在项目下分配，计入项目进度。

**Q：留言 @ 了经理没收到钉钉？**  
A：确认经理人员档案已绑定 userid，且您 @ 的是该任务上的负责人、协办人或项目负责人。
