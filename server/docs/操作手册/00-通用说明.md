# 通用说明（全部角色）

## 1. 登录与访问

### 正式使用（推荐）

1. 打开钉钉 → 工作台 → 找到 **恒慧管** 应用
2. 系统自动完成钉钉免登，无需输入账号密码
3. 首次使用需管理员在「人员档案」中同步钉钉通讯录，绑定 `userid`

### 演示模式（开发/内测）

- 侧边栏底部可切换不同人员账号
- 推送不会真实发送到钉钉（或走演示队列）

### 登录失败常见原因

| 现象 | 处理方式 |
|------|----------|
| 提示未找到钉钉用户 | 联系管理员同步通讯录并绑定 userid |
| 数据空白 | 检查网络，刷新页面；确认服务端已启动 |
| 登录过期 | 重新从钉钉工作台打开应用 |

## 2. 工作台

所有角色工作台 **统一只展示本人相关任务**（负责人或协办人），包括：

- 进行中 / 待开始 / 已逾期 / 临期提醒
- 本人参与的常规任务与临时任务
- 工时汇总（预计工时、实际工时）

顶部告警条会在有 **逾期**（截止日期早于今天）或 **临期**（今天/明天到期）任务时自动提示。

> **延期与临期**：截止日期 **当天** 仍算 **临期**，不算已延期；只有截止日期 **已过**（早于今天）才显示为已延期。

---

## 3. 任务类型

| 类型 | 说明 |
|------|------|
| **常规任务** | 挂在项目下，可无限层级拆分子任务 |
| **临时任务** | 不关联项目，任何角色均可快速创建，用于零星事项 |

---

## 4. 任务状态

| 状态 | 含义 | 常见操作 |
|------|------|----------|
| 待开始 | 已分配，尚未执行 | 「开始执行」 |
| 进行中 | 正在推进 | 更新进度、「标记完成」 |
| 已完成 | 任务结束 | 可归档（经理/总经理） |
| 已暂停 | 临时搁置 | 编辑恢复 |
| 已驳回 | 被上级退回整改 | 修改后重新推进 |
| 已作废 | 不再执行 | 不参与进度统计 |
| 已归档 | 项目级归档后只读 | 仅查看 |

**注意**：有 **未完成的子任务** 时，上级任务不能直接「标记完成」。

---

## 5. 计划时间与工时

| 字段 | 说明 |
|------|------|
| 计划开始日期 | 手动填写 |
| 预计工时 | 手动填写（小时） |
| 计划截止日期 | **自动计算**（按服务端工作日历，跳过节假日/非上班日） |
| 实际工时 | **自动计算**（根据实际开始/结束日期与工作日历） |

工作日历由服务端统一配置，前端不可修改。详见 [工作日历配置](../工作日历配置.md)。

---

## 6. 修改留痕

编辑已有任务时 **必须填写「修改原因」**，否则无法保存。所有变更写入操作日志，可追溯。

---

## 7. 钉钉工作通知

绑定 `userid` 后，以下事件会自动推送钉钉工作通知：

| 事件 | 接收人 |
|------|--------|
| 新任务分配 | 任务负责人 |
| 任务转办 | 新负责人 |
| 任务驳回 | 原负责人 |
| 即将到期 | 任务负责人（系统自动扫描） |
| 任务逾期 | 任务负责人（系统自动扫描） |
| 任务完成 | **项目负责人** + **执行人直属上级** |
| 留言 @ 提及 | 被 @ 的任务负责人、协办人、项目负责人（需已绑定 userid） |

推送内容分块展示：**所属项目 → 任务详情 → 操作信息**。

未收到推送请检查：① 人员档案是否已绑定 userid；② 您是否为该事件的接收人（如完成通知发给项目负责人，不一定发给执行人本人）。

## 8. 文档、附件与留言

### 8.1 项目文档与任务附件

| 位置 | 谁能上传 | 说明 |
|------|----------|------|
| 项目详情 · 项目文档 | 总经理/管理员、本部门经理 | 需求书、方案等 |
| 任务详情 · 任务附件 | 负责人、协办人、经理、管理员 | 截图、交付材料等 |

**粘贴图片**：在任务详情 **附件区域** 点击后，按 **Ctrl+V** 可直接粘贴剪贴板中的图片（需已登录并连接服务端）。

需连接服务端；单文件大小限制见服务端配置（默认 20MB）。

### 8.2 任务留言讨论

任务详情底部 **留言讨论** 用于项目相关人员在任务上下文中沟通进度与问题。

| 项目 | 说明 |
|------|------|
| **谁可以留言** | 任务负责人、协办人、项目负责人；总经理/管理员；本部门经理（本部门任务） |
| **@ 提及** | 输入 `@` 弹出可选人员（负责人、协办人、项目负责人），选中后发送；被 @ 且已绑定 userid 的人员会收到钉钉 **留言提及** 通知 |
| **粘贴图片** | 在留言输入框内 **Ctrl+V** 粘贴图片，发送前可预览；**点击缩略图可放大查看** |
| **纯图片留言** | 可不填文字，仅发送图片 |

> 留言图片保存在留言记录中，不会混入任务附件列表。任务附件用于正式交付材料，留言用于过程沟通。

## 9. 常用快捷操作

| 操作 | 入口 |
|------|------|
| 快速建临时任务 | 侧边栏「快速建任务」或移动端底部「新建」 |
| 查看任务详情 | 任务中心 / 工作台 → 点击任务卡片 |
| 任务留言 / @ 提及 | 任务详情 → 留言讨论 → 输入 `@` 或 Ctrl+V 粘贴图片 |
| 粘贴任务附件图片 | 任务详情 → 附件区域点击后 Ctrl+V |
| 导出任务列表 | 任务中心页顶部「导出」 |
| 查看操作日志 | 侧边栏「操作日志」 |
