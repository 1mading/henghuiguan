# 项目文档与任务附件

## 存储结构

| 位置 | 内容 |
|------|------|
| `server/data/uploads/` | 文件二进制（按 fileId 命名） |
| `project.documents[]` | 项目文档元数据 |
| `task.attachments[]` | 任务附件元数据 |

元数据示例：

```json
{
  "id": "DOC-A1B2",
  "fileId": "F-ABC123",
  "name": "需求说明书.docx",
  "size": 102400,
  "mimeType": "application/vnd.openxmlformats-officedocument.wordprocessingml.document",
  "uploadedBy": "王元斌",
  "uploadedAt": "2026-06-16T08:00:00.000Z"
}
```

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/files/upload` | multipart: `file`, `entityType`(project/task), `entityId` |
| GET | `/api/files/:fileId` | 下载（需登录且有权限） |
| DELETE | `/api/files/:fileId` | 删除 |

## 权限

- **项目文档**：总经理/管理员、同部门经理可上传；项目可见成员可下载
- **任务附件**：负责人/协办人/部门经理/管理员可上传；任务可见成员可下载

## 限制

- 单文件最大 **20MB**（环境变量 `MAX_UPLOAD_MB` 可调）
- 支持：pdf, doc/docx, xls/xlsx, ppt/pptx, 图片, zip, txt, csv, md 等

## 前端展示

- **项目详情** → 「项目文档」区块
- **任务详情** → 「任务附件」区块（支持点击区域后 **Ctrl+V** 粘贴图片）
- **任务详情** → 「留言讨论」（**@ 提及**、**Ctrl+V 粘贴图片**、点击放大预览）
- `desc` 仍为简短文字摘要，长文档请上传为文件

## 留言图片

留言图片保存在 `comment.attachments[]`，与任务附件分开；发送前上传的文件暂存于 `task.commentPendingFiles`（服务端自动清理）。
