# 工作日历配置

工作日历由**服务端**统一管理，前端登录后通过 `/api/data/bootstrap` 拉取，不在页面内维护。

## 配置文件

- 默认模板：`server/src/data/work-calendar-default.json`
- 运行时数据：`server/data/henghuiguan.json` 中的 `workCalendar` 字段
- 服务启动时若库中无配置，会自动写入默认模板

## 修改方式

### 1. 直接编辑 JSON（推荐）

编辑 `server/data/henghuiguan.json` 的 `workCalendar` 后重启服务。

### 2. API（需管理员 token）

```http
PUT /api/work-calendar
Content-Type: application/json
Authorization: Bearer <token>

{
  "scheduleMode": "alternate",
  "referenceDate": "2026-01-05",
  "referenceIsSingleWeek": true,
  "hoursPerDay": 8.5,
  "workStartHour": 8.5,
  "workEndHour": 17,
  "holidays": ["2026-01-01"],
  "extraWorkdays": ["2026-02-14"]
}
```

### 3. 辅助计算 API

- `POST /api/work-calendar/calc-end-date` — `{ startDate, hours }`
- `POST /api/work-calendar/calc-actual-hours` — `{ startDate, endDate? }`

## 字段说明

| 字段 | 说明 |
|------|------|
| `scheduleMode` | `alternate` 大小周 / `single` 全年单休 / `double` 全年双休 |
| `referenceDate` | 大小周基准周的周一 |
| `referenceIsSingleWeek` | 基准周是否为单休周 |
| `workStartHour` | 上班时间（8.5 = 8:30） |
| `workEndHour` | 下班时间（17 = 17:00） |
| `holidays` | 法定节假日 |
| `extraWorkdays` | 调休补班日 |
