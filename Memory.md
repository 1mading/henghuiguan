# Memory.md · 项目记忆（P1）

## 核心目标

**恒慧管**：部门内部项目管控系统——项目/任务树、权限、协作、钉钉集成、更新记录。

代码与资源位于仓库根目录下的 **`恒慧管项目/`**。

## 角色

| 代码 | 角色 |
|------|------|
| `gm` | 总经理 |
| `admin` | 管理员 |
| `manager` | 部门经理 |
| `staff` | 执行人员 |

查看与编辑分离（R-001）：全员可看全公司未归档项目；编辑仍按负责人/协办/经理范围。任务中心仍只显示本人任务。

## 阶段现状

主链路已落地：三级权限、无限层级任务树、临时任务、状态/转办/驳回/作废、饱和度、Excel 导入导出、JSON 存储与 bootstrap/sync、钉钉免登与工作通知（需配置）、项目文档/附件、操作面 Lite、系统更新记录。

细节与验收说明见 [`恒慧管项目/恒慧管-需求整理.md`](恒慧管项目/恒慧管-需求整理.md)。

## 关键决策

| 决策 | 说明 |

| 根目录文件体系 | 仅 AI 文档 + `.ai` + `恒慧管项目/`；工具目录保留 |
