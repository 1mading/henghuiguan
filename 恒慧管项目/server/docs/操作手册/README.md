# 恒慧管 · 操作手册（按角色）

> 部门内部项目管控系统 · 正式环境从钉钉工作台打开  
> 访问地址示例：`https://henghuiguan.handagroup.com/app`

---

## 角色说明

| 角色 | 系统标识 | 定位 | 数据范围 |
|------|----------|------|----------|
| **总经理** | `gm` | 全局统筹、审核决策 | 全公司所有项目与任务 |
| **管理员** | `admin` | 系统运维、与总经理同等权限 | 全公司所有项目与任务 |
| **部门经理** | `manager` | 本部门项目落地、任务分配与监督 | 全公司项目只读；本人负责/创建的项目可管理 |
| **执行人员** | `staff` | 任务执行、进度反馈 | 仅本人负责/协办的任务与相关项目 |

> **说明**：总经理与管理员在系统中权限相同（`isFullAccess`），菜单与操作一致，区别仅在岗位职责。

**最近更新（2026-07）**：操作面 Lite——「我的待办」合并工作台与任务中心；导航收为待办/项目/新建/更多；详情主操作开始/完成/留言；编辑高级字段折叠。**权限、协办审批、强制改因、饱和度计算等业务逻辑未改。**

**此前更新（2026-06）**：任务协办人（告知性/辅助性）、每日投入工时、跨项目任务依赖、滚动大屏；留言 @、粘贴图片、钉钉文档；全员只读浏览项目；项目团队成员、系统更新提醒、任务优先级。
